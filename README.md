# Bot style

## Attachments:
Attach with message (do not split) at any time

## Simple message:


<ins> >>  **Tem // NTheme**  << </ins>  
  
Text example. Here is a simple joke: A cake is a Lie...

## Reply:
At <<-->> should be '  ' (2 spaces) at the beginning of each line


<ins> >>  **Pelmen**  << </ins>  
  
But where should I laugh?
  
<<--------------->>  
  <ins> >>  _In reply to message by_ **Tem // NTheme**  << </ins>  
  
  Text example. Here is a simple joke: A cake is a Lie...  
>>---------------<<  

## Reply to attachment:
Resend always and reply


<ins> >>  **Tem // NTheme**  << </ins>  
  
Here, 'cause it's a joke!
  
<<--------------->>  
  <ins> >>  _In reply to attachment-type by_ **Tem // NTheme**  << </ins>  
>>---------------<<  

## (Perfect) Forwarding
Add 2 spaces at every new level of forwarding in <<-->>


<ins> >>  **Tem // NTheme**  << </ins>  
  
My joke is dead...(

<<--------------->>  
  <ins> >>  _Forwarded from_ **Tem // NTheme**  << </ins>
  
  Text example. Here is a simple joke: A cake is a Lie...
  
<>---------------<>  
  <ins> >>  _Forwarded from_ **Tem // NTheme**  << </ins>
  
  Here, 'cause it's a joke!
  
  <<--------------->>  
    <ins> >>  _Forwarded from_ **Pelmen**  << </ins>
    
    But where should I laugh?
  >>---------------<<  
>>---------------<<
